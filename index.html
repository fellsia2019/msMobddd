Проект собирает в себе лучшее на рынке сотовых операторов: удобный высококонверсионный сайт.

Проект на 1С:Bitrix в связке с Vue.js

Это позволяет моментально отражать десятки тысяч номеров, фильтровать их и показывать клиенту.

UI/UX Elements

Технологичность проекта — наша гордость. В момент заказа номера автоматически бронируются через API у оператора, чтобы другие партнеры не могли продать номер.

Автоматическая выгрузка тарифов. Работа над сайтом требует постоянных усилий — тарифны и планы операторов обновляются ежедневно. Реализована система, автоматически собирающая все обновления. Это полностью автономная система, не требующая обновления контента. Экономия в тысячи человеко-часов.
